Error code Description
E0 PalmdaClient is already defined. Existing version: '${existingVersion}' and this version: '${version}'.

There is already a PalmdaClient variable in the page (window). palmda-client.js may be loaded more than once.
E1 Origin is empty!

The iFrame must be opened with '#origin=XXX' hash. This hash is added to the iFrame's src by Palmda.
E2 The constructor is private, please use instance() method.

PalmdaClient is a singleton, use PalmdaClient.instance() to get the instance.
E3 Origins does not match. Expected '${expectedOrigin}' but found '${realOrigin}'!

The message is not targeted to this page. This error is not expected to be seen in modern browsers because they block unrelated messages themselves.
E4 Event data (event.data) is missing.

Palmda requires all events to contain data (at least data.type)
E5 Event data type (event.data.type) is missing.

Palmda requires all events to contain data and data.type.
E6 PalmdaClient is not defined. Call Plugin Library should be added to the page after PalmdaClient is defined.

PalmdaClient is not defined. Call Plugin Library should be added to the page after PalmdaClient is defined.
