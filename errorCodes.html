Error Reference

Error code Description
E0 GrispiClient is already defined. Existing version: '${existingVersion}' and this version: '${currentVersion}'.

There is already a GrispiClient variable in the page (window). grispi-plugin.js may be loaded more than once.
E1 Origin is empty!

The iFrame must be opened with '#origin=XXX' hash. This hash is added to the iFrame's src by Grispi.
E2 The constructor is private, please use instance() method.

GrispiClient is a singleton, use GrispiClient.instance() to get the instance.
E3 Origins does not match. Expected '${expectedOrigin}' but found '${realOrigin}'!

The message is not targeted to this page. This error is not expected to be seen in modern browsers because they block unrelated messages themselves.
E4 Event data (event.data) is missing.

Grispi requires all events to contain data (at least data.type)
E5 Event data type (event.data.type) is missing.

Grispi requires all events to contain data and data.type.
E6 GrispiClient is not defined. Call Plugin Library should be added to the page after GrispiClient is defined.

GrispiClient is not defined. Call Plugin Library should be added to the page after GrispiClient is defined.
E7 GrispiCallClient is already defined. Existing version: '${existingVersion}' and this version: '${currentVersion}'

You have added the grispi-call-plugin.js file more than once.
E8 Following methods are not not implemented.\n${missingMethods}\nImplement them via 'GrispiClient.prototype.call.<methodName> = aFunction'

You need to implement client's required methods.
E9 pluginId is empty!

The iFrame must be opened with '#pluginId=XXX' hash. This hash is added to the iFrame's src by Grispi.
E10 iframeAuth is empty!

The iFrame must be opened with '#iframeAuth=XXX' hash. This hash is added to the iFrame's src by Grispi.
