Grispi Right Panel Apps

Grispi Right Panel Apps are web applications with their own web UI which are rendered in iframe in Grispi pages.

Developing a Right Panel App for Grispi

Follow the steps:
  1. Create a web application with your preferred technology stack.
    • Your web pages should be responsive.
  2. Load grispi-app.js in your web page.
  3. Make sure you initialize the Grispi object as shown here.
  4. Create an app registration request to Grispi Help team from this support form.

Using ngrok for development

ngrok is a free utility which opens your localhost to the internet with SSL and (random) domain name.

When you create your Grispi App and run it on localhost, in order to test it you need to give a publicly reachable URL in your manifest.json so that grispi.com or grispi.net be able to load it in your users' browsers.

When you use ngrok with following command: ngrok http 8080 it gives you a URL with a random domain.
Whoever makes to request to this URL, ngrok forwards the request to your app running on your localhost.
